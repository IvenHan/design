Golang 学习笔记 01

go语言由包构成 Go 语言是由『包』构成的，package main 创建一个叫做『main』的包。一个包可以引入其他的包，如上述实例中的import "fmt",在『main』包中引入了『fmt』的包。引入后，可以在函数体中调用这个包中的函数。 fmt 是一个go语言自带的包 package main import "fmt" func main() { i := 1 var c int fmt.Println(i + c) } 包的入口函数 func main(){……}是这个包的入口主函数，使用这个包从这个主函数调用开始。 变量的定义 var i int a := 100 b = a + i 在go语言中，一般用var v type的形式来声明变量，在函数内可以用简化的方式v := value的方式来声明
